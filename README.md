# back_quelepaso
Qué le pasó?  Es una plataforma Inteligente de Diagnóstico y Resolución de Problemas del Hogar es una aplicación móvil diseñada específicamente para el contexto colombiano, cuyo objetivo es ayudar a los usuarios a identificar, comprender y resolver problemas domésticos frecuentes de manera rápida, económica y segura.
